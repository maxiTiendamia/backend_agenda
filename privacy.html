Política de Privacidad

Esta aplicación utiliza la API de WhatsApp de Meta para automatizar la gestión de reservas de turnos para negocios. Respetamos tu privacidad y nos comprometemos a proteger tus datos personales.

Recopilación de datos

No almacenamos información sensible de los usuarios. Los datos utilizados (como número de teléfono o nombre) son procesados únicamente para agendar reservas y no se comparten con terceros.

Uso de la información

Los datos se usan únicamente para ofrecer el servicio de agenda automatizada vía WhatsApp, integrándose con Google Calendar para verificar y agendar turnos disponibles.

Seguridad

La información transmitida está protegida mediante conexiones seguras (HTTPS) y solo se utiliza dentro del flujo de automatización entre WhatsApp y Google Calendar.
